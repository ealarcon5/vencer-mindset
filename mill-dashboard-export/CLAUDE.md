## How to Update the Dashboard

When asked to update the dashboard:
1. Edit `index.html` directly — all content lives in one file
2. Match existing dark style exactly: `#080808` bg, `#C9A87C` gold accent, card structure
3. Keep all four sections: Focus Bar, Stats Row, Grid cards, Footer
4. Update footer: `Updated by Claude — [today's date]`
5. Commit and push to `main`

**Dashboard sections and their purpose:**
- **Focus Bar** → single needle-mover for today, updated daily
- **Stats Row** → live metrics: Vencer Co AOV / Vencer Baseball clients / Creative Rise members / posts done
- **Must Do Today** → 3–5 checkbox items, highest priority first
- **Vencer Co Roadmap** → phase tracker toward Meta ads
- **Vencer Baseball Priorities** → phase tracker for facility growth
- **Creative Rise Society** → member count + phase tracker
- **Today's Content** → platform-by-platform content checklist (pull from Words on the Wrist skill)
- **Personal Brand** → world-building phase tracker (references vencer-mindset repo)
- **Workout** → Day A/B split, toggleable
- **Series Tracker** → Words on the Wrist episode progress
